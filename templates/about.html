Project Overview

This application uses machine learning algorithms to predict the risk of post-operative complications for thoracic cancer surgery patients. The system analyzes various patient parameters to provide a personalized risk assessment.

The primary goal is to assist thoracic surgeons and oncologists in making informed decisions about patient care, surgical approaches, and post-operative management strategies.

Predictive Model

Machine Learning Approach

Logistic Regression and Random Forest as the main machine learning models, with hyperparameter tuning and evaluation to select the best performing model.

  • Random Forest: This Algorithms to improve prediction performance,robustness,Handles Non-linearity,and Feature importance
  • Linear Regression:This Algorithm provides Interpretability,efficiency,probability estimates and compatability
Data Sources

The model is trained on the Thoracic Surgery Dataset, which contains real patient data collected from post-operative thoracic surgery patients. The dataset includes various clinical parameters and outcomes.

Performance Metrics

The model's performance is evaluated using various metrics:

  • Accuracy: Overall correctness of predictions
  • Sensitivity: Ability to correctly identify high-risk patients
  • F1-Score: Ability to balance precision and recall(model performance)
  • ROC-AUC: Discriminative ability of the model across different thresholds

Clinical Relevance

Post-operative complications following thoracic cancer surgery can significantly impact patient outcomes and quality of life. By predicting these risks before surgery, clinicians can:

  • Implement targeted pre-operative optimization strategies
  • Adjust surgical approaches to minimize risk
  • Prepare appropriate post-operative care plans
  • Have more informed discussions with patients about risk-benefit considerations
Important Note: This tool is designed to assist clinical decision-making, not replace it. The predictions should be considered alongside comprehensive clinical evaluation and professional judgment.
